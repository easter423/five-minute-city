# 거리 파사드 에셋 크레딧

## City Mega Pack
- **작가**: GrafxKid
- **라이선스**: CC0 1.0 (Public Domain) — 저작자 표시 의무 없음, 상업적 사용 가능
- **출처**: https://opengameart.org/content/city-mega-pack
- **파일**: `CITY_MEGA.png` (1952×2800)

## 사용 부위
시트의 "Main buildings layer"(세로 ~1640–1840 구간)에 있는 사이드뷰 스토어프론트
파사드를 잘라 five-minute-city 거리의 named building 8동 외관으로 사용한다.
좌표(sprite.sx/sy/sw/sh)는 `js/config.js` 의 `CITY_BUILDINGS` 에 정의.
